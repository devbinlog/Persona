# Moderation & Safety Rules

## AI
- No lyrics, no streaming, no hate/violence/sex/illegal/self-harm guidance.
- Violation => block + safe alternative.

## Lounge
- Ban: hate/harassment/sexual/illegal/personal info.
- Actions: warn/delete/limit/ban.

## Character Sim
- Auto moderation before save.
- Max turns limit + timeout.
- Save scene summary card.

## Protections
- word filter, rate limit, report flag, logs.
